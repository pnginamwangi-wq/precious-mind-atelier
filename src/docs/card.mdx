# Card

Stories: [_Luxury / Card_](?path=/story/luxury-card--default).

## When to use

| Variant | Use for |
| --- | --- |
| `default` | Chapter tiles, institute cards, feature grids. |
| `elevated` (gold shadow) | The single featured item in a grid, or a pricing tier. |
| `compact` | Sidebar summaries, dense dashboards. |

## Do

- Pair a card with one `Eyebrow`, one heading, one line of body, and at most one CTA.
- Use `Hairline` inside a card only to separate metadata from body — never as decoration.

## Don't

- Elevate every card. If everything glows, nothing does.
- Nest cards. If you need hierarchy, use a `Container narrow` inside a section instead.
